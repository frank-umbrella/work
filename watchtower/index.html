Settings

Master webhook URL

All per-agent webhook events (IP changes, backup failures) POST a JSON payload to this URL when their per-PC webhookEnabled toggle is on. Set once for the whole fleet.

Whether each agent fires events to this URL is controlled by the per-PC Webhook on IP change toggle in that host's drawer. Leaving the master URL empty silences all webhook events fleet-wide.
