Status icon previews

Six options for the consolidated health pill in the Endpoints table Status column. Each option shows all four states (Healthy / Warning / Critical / Muted) in both light and dark backgrounds, and each pill is rendered with the exact same CSS that ships in production -- this is what you'll see in the dashboard. Pick by number and reply with the choice; I'll wire it in.

My recommendation

Option 3 (Heroicons solid). Reasons: pixel-identical rendering on every OS / browser, clear at the 14px size we use, polished without being playful, matches the visual language of every other modern MSP / DevOps tool. Filled glyphs read better than outlines at small sizes. Option 5 (the ops shield set) is a strong second if you want a more "this is serious infrastructure" feel.
