Umbrella Automation · Brand Guidelines

Our brand,
kept simple and consistent.

Use this page to grab the official Umbrella Automation logo files, colors, and typography — and to make sure they get used the right way. If you’re designing something with our brand on it, this is the source of truth.

Logo files render with Eurostile-Black where available and a close geometric fallback otherwise.

Logo

Four logos, one identity.

Pick the variant that fits the space you’ve got. The primary lockup is the default — reach for the others only when it doesn’t fit or read clearly.

Primary logo

SVG

Horizontal lockup, full color. The default mark for almost every situation: web headers, business cards, vehicle wraps, slide decks.

Stacked logo

SVG

Vertical lockup. Use when the primary is too wide — square ad units, narrow sidebars, badge pockets on shirts and hats.

Wordmark

SVG

Type only. Use when the submark is shown elsewhere on the same surface, or when the icon doesn’t add value (e.g. inside a paragraph signature).

Submark

SVG

Icon only. Use for app icons, favicons, social avatars, and any tight square where the wordmark wouldn’t be readable.

On dark backgrounds

SVG

Reversed treatment for dark surfaces. The house outline becomes white, the AUTOMATION wordmark goes white, and the brand blue stays put.

Single color

Dark Light

For one-color print, embroidery, vinyl cut, etched signage, or anywhere full color isn’t available. Pick the version with the strongest contrast against the surface.

Clear space

Give the logo room to breathe.

Always leave clear space around the logo equal to X — the height of the submark icon. Nothing else (text, photos, edges, other logos) should encroach on this zone.

X X X X
X = the height of the submark.
Measure from the apex of the roof to the bottom of the icon.

Minimum size

Don’t make it too small to read.

Below these sizes the wordmark stops being legible and the icon starts to muddy. If you can’t make the size, use a larger version on a different surface.

Primary & stacked logo

120 px wide
(≈ 1 in / 25 mm in print)

Below this width the wordmark gets too thin to read at a glance, especially on screens with subpixel rendering.

Submark

32 px wide
(favicon & chip-sized)

The submark holds together down to favicon size. For 16-px favicons, drop the circuit dots to two and bump the stroke weight if it looks fragile.

Typography

Eurostile-Black, set in all caps.

The wordmark is built from Eurostile-Black. Wide, geometric, slightly rounded — the personality is “modern but grounded.” If you need a free fallback for the web, Saira 900 is the closest match.

UMBRELLA
AUTOMATION

  • FamilyEurostile-Black
  • StyleRegular
  • Size50 pt
  • Leading38 pt
  • Tracking25 (≈ 0.025 em)
  • Vertical scale100%
  • Horizontal scale100%
  • Baseline shift0 pt
  • CaseAll Caps (set in textbox, not typed)

Note: Eurostile is a licensed font. The downloadable SVGs reference it by name — viewers without the font installed will see the Saira fallback. For pixel-perfect rendering in print or signage, install Eurostile-Black on your machine and re-open the SVG, or have your designer convert the wordmark to outlines before exporting.

Gradient Overlays

Three gradient overlays carry the brand color story. All use Normal blend mode, 100% opacity, Reverse on, Align with Layer on, and 150% scale — they differ in style, angle, and stops.

UMBRELLA wordmark

  • StyleReflected
  • Angle90°
  • ReverseOn
  • Stops5 stops, light → bright → deep → bright → light

AUTOMATION + house outline

  • StyleLinear
  • Angle90°
  • ReverseOn
  • Stops#C8C8C8 → #6E6E6E (named “White”)

Circuit lines

  • StyleLinear
  • Angle−90°
  • ReverseOn
  • Stops#5FC1EC → #177CB8 (named “Custom”)

CSS recipes

Bevel & emboss can’t be reproduced perfectly in CSS, but the gradient overlays translate cleanly using background-clip: text. Drop these into a stylesheet and apply to a heading element to render the wordmark with the right gradient on the web.

/* UMBRELLA — reflected blue, top & bottom highlight */
.umbrella {
  background: linear-gradient(
    to bottom,
    #a4d6ed 0%,
    #26a6e6 18%,
    #0c5e9c 50%,
    #26a6e6 82%,
    #5fc1ec 100%
  );
  -webkit-background-clip: text;
  background-clip: text;
  color: transparent;
  font-family: "Eurostile", "Saira", system-ui, sans-serif;
  font-weight: 900;
  letter-spacing: 0.025em;
  text-transform: uppercase;
}

/* AUTOMATION — light-to-dark gray (also used for house outline stroke) */
.automation {
  background: linear-gradient(to bottom, #c8c8c8 0%, #6e6e6e 100%);
  -webkit-background-clip: text;
  background-clip: text;
  color: transparent;
  font-family: "Eurostile", "Saira", system-ui, sans-serif;
  font-weight: 900;
  letter-spacing: 0.025em;
  text-transform: uppercase;
}

/* Circuit lines — bottom-to-top blue (used as SVG linearGradient or CSS bg) */
.circuit {
  background: linear-gradient(to top, #5fc1ec 0%, #177cb8 100%);
}
