Umbrella LogMeIn Installer

Three short steps.
The download should start automatically — if it didn't, use the button below.

Download installer Umbrella-Slim.exe · Windows

1Find the file you just downloaded

Pick your browser to see exactly where it landed:

Chrome shows downloads in a small tray icon (↓) in the top-right toolbar, just left of your profile picture. Click it to see Umbrella-Slim.exe.

Don't see the tray? Press Ctrl + J to open the full Downloads page, or open File Explorer → Downloads folder.

Edge shows the same downloads tray (↓) in the top-right. Click it to find Umbrella-Slim.exe, then choose Open file.

Or press Ctrl + J, or open File Explorer → Downloads.

Firefox puts a downward arrow icon in the top-right toolbar. Click it to see the file and pick Show in Folder.

Or press Ctrl + J for the full Downloads library.

Safari shows a downloads arrow in the top-right of the toolbar (next to the share button). The popover lists the file with Show in Finder.

On macOS the file lands in ~/Downloads. (Note: Umbrella-Slim.exe is a Windows installer — you'll need to run it on a PC.)

2Right-click and Run as administrator

Right-click Umbrella-Slim.exe and choose Run as administrator. When Windows asks “Do you want to allow this app to make changes?”, click Yes.

3Click Install, then email us the info file

The installer window opens — click Install. When it finishes, a file called Computer-Info.txt opens automatically and a copy is saved to your Desktop.

Send us that file (or paste its contents) so we can finish setting you up.
