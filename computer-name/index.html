Find Your Computer Name

A quick, illustrated guide to locating your computer's name — the bit of info our team often needs when setting up or troubleshooting your device.

UMBRELLAAUTOMATION
💻

Why are we asking for this? Every computer has a unique name (sometimes called the “device name” or “hostname”). It helps us find your exact machine when we set up software, fix an issue, or add you to the network. Pick your operating system below and follow the steps — it only takes a few seconds.

Fastest

Press ⊞ Windows + I to open Settings → System → About. Your name is at the top under Device name.

  1. Open Settings
    Press ⊞ Windows + I together, or click the Start button and choose the Settings gear.
  2. Go to System → About
    System is usually selected at the top of the left sidebar. Scroll to the bottom of the System list and click About.
  3. Read your Device name
    At the top of the About page you'll see Device name. That's your computer name. Click Copy to grab it exactly.
    Settings — About
    System › About
    DESKTOP-7K2QX9P
    HP ZBook Firefly 14 G8
    Rename this PC
    i Device specifications Copy
    Device nameDESKTOP-7K2QX9P
    ProcessorIntel Core i7-1165G7
    Installed RAM16.0 GB

    Your name will look different — e.g. DESKTOP-7K2QX9P, FRONT-DESK-PC, or LAPTOP-J3.

Alt Prefer a quick command? ›
  1. Press ⊞ Windows + R, type cmd, and press Enter.
  2. Type hostname and press Enter.
  3. The single word printed is your computer name.
Command Prompt
C:\Users\You> hostname
DESKTOP-7K2QX9P
Fastest

Press ⊞ Windows + I to open Settings → System → About. Look for Device name under “Device specifications.”

  1. Open Settings
    Press ⊞ Windows + I, or click Start → the Settings gear.
  2. Click System
    In the Settings window, click System (the laptop-and-display icon).
  3. Scroll down to About
    In the left sidebar, scroll all the way down and click About.
  4. Find Device name
    Under the Device specifications heading, the first line is Device name — that's your computer name.
    Settings — About
    i Device specifications Copy
    Device name OFFICE-PC-04
    Full device nameOFFICE-PC-04
    ProcessorIntel Core i5-8250U
    Installed RAM8.00 GB
    Rename this PC

    Yours might read OFFICE-PC-04, DESKTOP-AB12CDE, or a custom name your IT team set.

Alt Right-click method ›
  1. Right-click the Start button (bottom-left corner).
  2. Click System in the menu that appears.
  3. The About page opens — read Device name.
Alt Quick command ›
  1. Press ⊞ Windows + R, type cmd, press Enter.
  2. Type hostname and press Enter — the word shown is your computer name.
Command Prompt
C:\Users\You> hostname
OFFICE-PC-04
Fastest

Click the Apple menu () → System Settings → General → About. Your computer name is the Name at the top.

  1. Open the Apple menu
    Click the Apple logo () in the very top-left corner of your screen.
  2. Choose System Settings
    Click System Settings… (on macOS Monterey and earlier this is called System Preferences).
  3. Go to General → About
    In the sidebar click General, then click About.
  4. Read the Name
    The very first row, Name, is your computer's name.
    System Settings — About
    General › About

    macOSSequoia 15.5
    ChipApple M3
    Memory16 GB
    Serial numberC02·······Q

    Yours will look like Reception-MacBook-Pro or Johns-iMac.

Alt Using the Terminal ›
  1. Open Terminal (press ⌘ Cmd + Space, type Terminal, press Return).
  2. Type scutil --get ComputerName and press Return.
  3. The line printed is your computer name. (hostname also works on any Mac version.)
Terminal — zsh
you@mac ~ % scutil --get ComputerName
Reception-MacBook-Pro

⚡ Works on any version

No matter which Windows or Mac you have, you can always open the command line and type one short word:

hostname
> hostname
YOUR-COMPUTER-NAME

On Windows that's Command Prompt (⊞ Win + R → cmd). On Mac it's Terminal. Type hostname, press Enter, and copy the result.
